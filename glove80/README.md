# Glove80 Keyboard Layout

This config is constantly changing, so the linked config here might be outdated.
My current keyboard layout including combo keys and mouse simulation for the glove80 is available [here](https://my.moergo.com/glove80/#/layout/user/bc424adc-7505-4ff6-8e49-8b27ff5ce51c)

I recently switched from `df` and `jk` as shift combo keys to hold-tap shift on `backspace` and `space`.
The delay disappears for space during writing and only is noticeable for the backspace if there was a longer pause between the last input and the backspace.
